# 来只插件图片功能说明

## 🎉 新功能概述

来只插件现在支持真实的图片存储和管理功能！你可以下载URL图片到本地，按不同来只名称分文件夹存放，并随机发送图片。

## 📁 存储结构

```
data/
└── images/
    ├── 猫咪/
    │   ├── image_1.jpg
    │   ├── image_2.png
    │   └── ...
    ├── 狗狗/
    │   ├── photo_1.jpg
    │   └── ...
    └── 风景/
        ├── view_1.jpg
        └── ...
```

每个来只都有自己独立的图片文件夹，图片按名称分类存储。

## 🚀 使用方式

### 1. 添加图片（支持URL下载）

```
添加猫咪 https://example.com/cat.jpg
```
- 自动下载图片到本地
- 存储到 `data/images/猫咪/` 文件夹
- 更新数据库计数

**支持多种图片格式**: JPG, PNG, GIF, WebP, BMP

### 2. 随机获取图片

```
来只猫咪
```
- 从猫咪文件夹中随机选择一张图片
- 直接发送图片到聊天
- 显示来只信息

### 3. 查询图片信息

```
查询猫咪
```
- 显示来只的详细信息
- 显示数据库计数和实际文件数量
- 包含创建时间、别名等

### 4. 删除来只（包括图片）

```
删除猫咪
```
- 删除来只记录
- 删除所有相关图片文件
- 释放磁盘空间

### 5. 查看所有来只

```
/列表
```
- 显示所有来只列表
- 显示每个来只的图片计数
- 显示实际文件数量

## 🎯 核心功能

### 图片下载
- ✅ **URL下载**: 支持HTTP/HTTPS链接
- ✅ **格式支持**: JPG, PNG, GIF, WebP, BMP
- ✅ **自动命名**: 智能文件名处理
- ✅ **重名处理**: 自动添加序号避免覆盖

### 文件管理
- ✅ **分文件夹存储**: 每个来只独立文件夹
- ✅ **路径安全**: 自动过滤不安全字符
- ✅ **批量删除**: 删除来只时清理所有图片

### 随机发送
- ✅ **真正随机**: 每次可能不同
- ✅ **图片发送**: 直接发送到聊天
- ✅ **错误处理**: 无图片时显示提示信息

## 📊 计数机制

插件维护两种计数：

1. **数据库计数**: 快速统计，每次添加+1
2. **实际文件数**: 实时扫描文件夹获取

```
查询猫咪
→ 图片数量: 10 (实际文件: 8)
```

这帮助你了解数据库和实际文件的同步状态。

## 🔧 技术细节

### 存储位置
- **图片路径**: `data/images/<来只名称>/`
- **数据库文件**: `data/laizhi_db.json`

### 文件命名规则
- **保留原文件名**: 从URL提取
- **自动生成**: 无文件名时使用时间戳
- **重名处理**: 自动添加序号 `_1`, `_2` 等

### 安全特性
- **路径过滤**: 移除不安全字符
- **文件验证**: 检查文件扩展名
- **错误处理**: 下载失败不影响数据库

## 💡 使用场景

### 1. 收集表情包
```
新建表情包
添加表情包 https://example.com/funny1.jpg
添加表情包 https://example.com/funny2.jpg
来只表情包  # 随机发送表情包
```

### 2. 图片分类管理
```
新建猫咪
新建狗狗
添加猫咪 https://cat.example.com/cute1.jpg
添加狗狗 https://dog.example.com/puppy1.jpg
/列表  # 查看所有分类
```

### 3. 随机图片分享
```
新建壁纸
添加壁纸 https://wallpaper.example.com/1.jpg
添加壁纸 https://wallpaper.example.com/2.jpg
来只壁纸  # 随机分享壁纸
```

## 🛡️ 错误处理

### 下载失败
```
添加猫咪 https://invalid-url.jpg
→ ❌ 图片下载失败: https://invalid-url.jpg
```

### 来只不存在
```
来只不存在的来只
→ ❌ 未找到来只 '不存在的来只'
```

### 无图片情况
```
来只空来只
→ 💡 提示: 该来只暂无图片，使用 '添加空来只 <图片URL>' 来添加图片
```

## 🔄 与原有功能的兼容性

✅ **完全兼容**: 新的图片功能不影响原有的文本计数功能
✅ **别名支持**: 所有图片功能都支持别名查询
✅ **数据库同步**: 图片计数和数据库计数同步更新
✅ **平滑升级**: 现有数据无需迁移，自动启用新功能
