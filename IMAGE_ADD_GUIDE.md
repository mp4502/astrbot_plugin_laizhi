# 📸 来只插件图片添加功能说明

## 🎯 功能概述

**来只插件现在支持从消息中的图片直接添加到来只图库！**

你不再需要手动输入URL，只需发送图片然后使用添加命令即可。

## 🚀 使用方式

### 方法1：发送图片后添加（推荐）

#### 步骤：
1. **发送一张图片**到聊天
2. **回复该图片**（或直接在下一行）发送命令：`添加猫咪`
3. **插件自动提取**图片并到来只图库

#### 示例：
```
[你发送了一张猫咪图片]
添加猫咪

✅ 为 '猫咪' 添加图片成功！
📁 已保存到: data/images/猫咪/cat.jpg
当前总数: 1
```

### 方法2：URL添加（仍然支持）

如果你有图片的URL链接，也可以直接添加：

```
添加猫咪 https://example.com/cute_cat.jpg
```

## 🔧 工作原理

### 图片识别机制

插件会自动检查消息中的所有组件，寻找包含图片的组件：

```python
for comp in messages:
    # 检查图片URL
    if hasattr(comp, 'url') and comp.url:
        image_url = comp.url
        break
    # 检查本地文件
    if hasattr(comp, 'file') and comp.file:
        image_url = comp.file
        break
```

### 自动下载和存储

找到图片后，插件会：
1. ✅ **下载图片**到本地 `data/images/<来只名称>/` 文件夹
2. ✅ **自动命名**，避免文件名冲突
3. ✅ **更新计数**，同步数据库
4. ✅ **返回结果**，显示保存位置

## 🛡️ 错误处理

### 情况1：来只不存在
```

❌ 来只 '不存在的来只' 不存在，请先使用 '新建不存在的来只' 创建！
```

### 情况2：未找到图片
```
添加猫咪

```

```

❌ 图片下载失败: [图片URL]
```
